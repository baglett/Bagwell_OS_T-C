1. Program Name

Bagwell OS SMS Assistant — a personal AI assistant service that communicates via SMS text messaging.

2. Program Description

Bagwell OS is a personal AI assistant that you interact with by sending and receiving SMS text messages. The service can manage tasks, send daily digest summaries, deliver notifications about scheduled events, and provide system status updates. This is a personal, non-commercial service operated by a single individual.

3. Message Frequency

Message frequency varies based on your usage. You may receive:

  • Conversational replies — one response per message you send to the service
  • Daily digests — up to one summary message per day (if configured)
  • Task reminders — as scheduled by you
  • System notifications — for events such as completed tasks, pull request updates, or system alerts

Non-urgent notifications are rate-limited to a maximum of one message per 60 seconds. Urgent notifications (system alerts, blockers) may bypass this limit.

4. Message and Data Rates

Message and data rates may apply. Standard messaging rates from your wireless carrier apply to all SMS messages sent to and received from this service. Contact your carrier for details about your messaging plan.

5. Opt-In

By texting the Bagwell OS phone number, you consent to receive SMS messages from this service. Only phone numbers that have been manually added to the service’s allowed list by the operator will receive messages.

6. Opt-Out

You can stop receiving messages at any time by texting STOP to the Bagwell OS phone number. You will receive a confirmation message and no further messages will be sent. To resume the service, text START.

7. Help
